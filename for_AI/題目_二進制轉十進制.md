# 二進制轉十進制 (Binary to Decimal)

## 問題描述

請寫一個程式，輸入一串只包含 0 和 1 的二進制字串，將它轉換成十進制整數並輸出。

## 題目說明

將二進制轉換為十進制的方法是：從最右邊的位元開始，每個位元的值乘以 2 的對應次方後加總。例如二進制 1101，從右到左依序為第 0、1、2、3 位，計算為 1×2⁰ + 0×2¹ + 1×2² + 1×2³ = 1+0+4+8 = 13。輸入保證是合法的二進制字串，不會有前導零（僅 "0" 本身例外）。

## 範例格式

| 使用者輸入 | 輸出 | 範例說明 |
|-----------|------|---------|
| 1101 | 13 | 1×2³ + 1×2² + 0×2¹ + 1×2⁰ = 8+4+0+1 = 13 |
| 1000 | 8 | 1×2³ + 0+0+0 = 8 |
| 1 | 1 | 1×2⁰ = 1 |
| 0 | 0 | 0×2⁰ = 0 |
| 11111111 | 255 | 8 個位元全為 1，= 2⁸-1 = 255 |

## 輸入格式規定

- 第一行：一串只包含 0 和 1 的字串（長度 1 ≤ L ≤ 10）

## 輸出格式規定

- 第一行：對應的十進制整數

## 核心概念

- 字串走訪（逐字元處理）
- 2 的次方計算（位權概念）
- 由左至右累加（每次乘以 2 的技巧）
