# 十進制轉二進制 (Decimal to Binary)

## 問題描述

請寫一個程式，輸入一個非負整數，將它轉換成二進制並輸出。

## 題目說明

二進制是以 2 為基底的進位制，每一個位數只有 0 和 1。將十進制數轉換為二進制的方法是：反覆將數字除以 2，記錄每次的餘數，最後將餘數由下而上排列，即為二進制結果。例如 13 ÷ 2 = 6 餘 1、6 ÷ 2 = 3 餘 0、3 ÷ 2 = 1 餘 1、1 ÷ 2 = 0 餘 1，由下而上排列得到 1101。特別規定：輸入 0 時，輸出 0。

## 範例格式

| 使用者輸入 | 輸出 | 範例說明 |
|-----------|------|---------|
| 13 | 1101 | 13 = 8+4+1 = 2³+2²+2⁰，對應二進制為 1101 |
| 8 | 1000 | 8 = 2³，對應二進制為 1000 |
| 1 | 1 | 1 = 2⁰，二進制仍為 1 |
| 0 | 0 | 特殊情況，0 的二進制為 0 |
| 255 | 11111111 | 255 = 2⁸-1，8 個位元全為 1 |

## 輸入格式規定

- 第一行：一個非負整數 N（0 ≤ N ≤ 1023）

## 輸出格式規定

- 第一行：N 的二進制表示（字串，不含前導零，僅 0 本身例外）

## 核心概念

- 除以 2 取餘數（迴圈）
- 字串串接與反轉
- 邊界條件處理（N = 0）
