# 智慧停車場管理員 (Smart Parking Manager)

## 問題描述

城市中一座多層立體停車場共有 M 層樓，每層有 N 個車位。管理系統會即時記錄每個車位的使用狀況。請寫一個程式，幫助管理員快速掌握整座停車場的空位分佈情形：計算空位總數，並找出哪一層樓的空位最多，方便引導車輛直接前往該層。

## 題目說明

停車場的每個車位以數字表示：**0 代表空位，1 代表已停車**。樓層編號由上而下從第 1 層開始。若有多個樓層的空位數相同且並列最多，則輸出**編號最小**的樓層。

## 範例格式

| 使用者輸入 | 輸出 | 範例說明 |
|-----------|------|---------|
| 3 4<br>1 0 1 0<br>0 0 0 1<br>1 1 0 0 | 7<br>2 3 | 共 3 層樓，每層 4 個車位。<br>第 1 層：位置 2、4 為空位，共 2 個空位<br>第 2 層：位置 1、2、3 為空位，共 3 個空位（最多）<br>第 3 層：位置 3、4 為空位，共 2 個空位<br>空位總數為 7，空位最多的是第 2 層，有 3 個空位 |
| 3 3<br>1 0 0<br>0 1 0<br>1 0 1 | 5<br>1 2 | 共 3 層樓，每層 3 個車位。<br>第 1 層：2 個空位<br>第 2 層：2 個空位（與第 1 層並列最多）<br>第 3 層：1 個空位<br>空位總數為 5，第 1 層與第 2 層並列最多，取編號最小的第 1 層，共 2 個空位 |
| 1 5<br>0 0 0 0 0 | 5<br>1 5 | 只有 1 層樓，5 個車位全部空著。<br>空位總數為 5，唯一的樓層即為第 1 層，有 5 個空位 |
| 2 4<br>1 1 1 1<br>1 1 1 1 | 0<br>1 0 | 所有車位均已停滿。<br>空位總數為 0，兩層都是 0 個空位，取編號最小的第 1 層 |

## 輸入格式規定

- 第一行：兩個正整數 M 和 N（1 ≤ M ≤ 10，1 ≤ N ≤ 10），以半形空格分隔，分別代表樓層數與每層車位數
- 第二行至第 M+1 行：每行輸入 N 個整數（0 或 1），以半形空格分隔，代表該層各車位的狀態

## 輸出格式規定

- 第一行：整座停車場的空位總數（整數）
- 第二行：空位最多的樓層編號，以及該層的空位數，兩者以半形空格分隔

## 核心概念

- 二維陣列的建立與走訪（逐行逐列讀取）
- 列（Row）加總
- 跨列比較找最大值（含並列處理）
